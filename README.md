# Graphic Design Feedback System

A collaborative design feedback platform using AJAX/PHP, JavaScript, and CSS.

## Features
- Pin-based contextual comments on design versions
- Threaded discussion threads for each pin
- Design version management
- Multi-project support
- Real-time feedback updates

## Tech Stack
- Backend: PHP
- Database: MySQL
- Frontend: JavaScript (AJAX), CSS
- Asset Storage: S3
